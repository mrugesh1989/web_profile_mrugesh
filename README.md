# Mrugesh Patel — Portfolio

Personal portfolio for **AI Data Engineering** roles. Static HTML/CSS/JS on [GitHub Pages](https://pages.github.com/).

**Live site:** https://mrugesh1989.github.io/web_profile_mrugesh/

## Deploy (GitHub Pages)

1. Open **Settings → [Pages](https://github.com/mrugesh1989/web_profile_mrugesh/settings/pages)**
2. Under **Build and deployment → Source**, choose **Deploy from a branch**
3. **Branch:** `main` · **Folder:** `/ (root)` · **Save**
4. Wait 1–2 minutes, then open the live URL above

No GitHub Actions workflow is required for this static site.

## Local preview

```bash
python3 -m http.server 8080
# http://localhost:8080
```
